# hub_news

A new Flutter application.

已经实现以下功能

1. 实况天气：网络请求相关
2. 相机：通过MethodChannel或者pub plugin的方式调用客户端原生硬件
3. 个人中心页：Flutter相关资料，github地址（用于求star）
4. 推送
